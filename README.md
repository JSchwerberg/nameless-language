# Nameless Language

## Develop

```sh
npm install
npm run server
# in a new tab
npm start
```

Open a new web page to `http://localhost:8000/`. Change any `.re` file in `src` to see the page auto-reload.

# Bundle for Production

Just run webpack. It will generate files used by `index.html`.

```sh
./node_modules/.bin/webpack
```
